Core classes¶

HL7apy - core classes

class hl7apy.core.Element(name=None, parent=None, reference=None, version=None, validation_level=None, traversal_parent=None)¶

Base class for all HL7 elements. It is not meant to be directly instantiated.

class hl7apy.core.Message(name=None, reference=None, version=None, validation_level=None, encoding_chars=None)¶

Class representing an HL7 message

Parameters:
  • name (basestring) – the HL7 name of the message (e.g. OML_O33)
  • reference – the reference structure (see hl7apy.load_reference())
  • version (basestring) – the HL7 version (e.g. “2.5”), or None to use the default (see hl7apy.set_default_version())
  • validation_level (int) – the validation level. Possible values are those defined in hl7apy.consts.VALIDATION_LEVEL class or None to use the default validation level (see hl7apy.set_default_validation_level())
  • encoding_chars (dict) – a dictionary containing the encoding chars or None to use the default (see hl7apy.set_default_encoding_chars())
to_er7(encoding_chars=None, trailing_children=False)¶

Returns the HL7 representation of the Element. It adds the appropriate separator at the end if needed

Parameters:encoding_chars – The encoding chars to use. If it is None it uses self.encoding_chars, which by default is hl7apy.consts.DEFAULT_ENCODING_CHARS
Return type:str the HL7 representation of the Element
to_mllp(encoding_chars=None, trailing_children=False)¶

Returns the er7 representation of the message wrapped with mllp encoding characters

Parameters:
  • encoding_chars (dict) – a dictionary containing the encoding chars or None to use the default (see hl7apy.set_default_encoding_chars())
  • trailing_children (bool) – if True, trailing children will be added even if their value is None
Returns:

the ER7-encoded string wrapped with the mllp encoding characters

class hl7apy.core.Segment(name=None, parent=None, reference=None, version=None, validation_level=None, traversal_parent=None)¶

Class representing an HL7 segment.

Parameters:
  • name (basestring) – the HL7 name of the segment (e.g. PID)
  • parent (an instance of hl7apy.core.Message, hl7apy.core.Group or None) – the parent
  • reference – the reference structure (see hl7apy.load_reference())
  • version (basestring) – the HL7 version (e.g. “2.5”), or None to use the default (see hl7apy.set_default_version())
  • validation_level (int) – the validation level. Possible values are those defined in hl7apy.consts.VALIDATION_LEVEL class or None to use the default validation level (see hl7apy.set_default_validation_level())
  • traversal_parent (an instance of hl7apy.core.Message, hl7apy.core.Group or None) – the temporary parent used during traversal
to_er7(encoding_chars=None, trailing_children=False)¶

Return the ER7-encoded string

Parameters:
  • encoding_chars (dict) – a dictionary containing the encoding chars or None to use the default (see hl7apy.set_default_encoding_chars())
  • trailing_children (bool) – if True, trailing children will be added even if their value is None
Returns:

the ER7-encoded string

 
>>> pid = Segment("PID")
>>> pid.pid_1 = '1'
>>> pid.pid_5 = "EVERYMAN^ADAM"
>>> print pid.to_er7()
PID|1||||EVERYMAN^ADAM

class hl7apy.core.SubComponent(name=None, datatype=None, value=None, parent=None, reference=None, version=None, validation_level=None, traversal_parent=None)¶

Class representing an HL7 subcomponent.

Parameters:
  • name (basestring) – the HL7 name of the subcomponent (e.g. CWE_1)
  • datatype (basestring) – the datatype of the component (e.g. ST)
  • value (basestring or instance of hl7apy.base_datatypes.BaseDataType) – the value of the subcomponent (e.g. ADT_A01)
  • parent (an instance of hl7apy.core.Component or None) – the parent
  • reference – the reference structure (see hl7apy.load_reference())
  • version (basestring) – the HL7 version (e.g. “2.5”), or None to use the default (see hl7apy.set_default_version())
  • validation_level (int) – the validation level. Possible values are those defined in hl7apy.consts.VALIDATION_LEVEL class or None to use the default validation level (see hl7apy.set_default_validation_level())
  • traversal_parent (an instance of hl7apy.core.Message, hl7apy.core.Group or None) – the temporary parent used during traversal
