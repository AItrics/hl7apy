MLLP Classes¶

class hl7apy.mllp.MLLPServer(host, port, handlers)¶

A TCPServer subclass that implements an MLLP server receives HL7 messages encoded in MLLP and redirects them to the correct handler.

Parameters:
  • host – the address of the listener
  • port – the port of the listener
  • handlers – a dictionary that specify the handler classes for every kind of supported message. The keys of the dictionary must be the message type and the values the handlers classes. The latter must be subclasses of AbstractTransactionHandler that implement the reply() method
class hl7apy.mllp.AbstractTransactionHandler(message)¶

Abstract transaction handler. Handlers should implement the reply() method which handle the incoming message.

Parameters:message – a Message object
reply()¶

Abstract method. It should implement the handling of the request message and return the response
