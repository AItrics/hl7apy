Validation module¶

HL7apy - validation

class hl7apy.validation.Validator(level)¶

Class that handles validation. It defines validation levels and validate an element using hl7apy.consts.VALIDATION_LEVEL.STRICT validation level

static is_quiet(level)¶

Check if the given validation level is quiet

Parameters:level (int) – validation level (see hl7apy.consts.VALIDATION_LEVEL)
Return type:bool
Returns:True if validation level is quiet
static is_strict(level)¶

Check if the given validation level is strict

Parameters:level (int) – validation level (see hl7apy.consts.VALIDATION_LEVEL)
Return type:bool
Returns:True if validation level is strict
static validate(element)¶

Checks if the hl7apy.core.Element is a valid HL7 message according to the official structures. It checks if it follows the max and min number of occurrences for every child and if it doesn’t contain children not allowed.

Parameters:element – hl7apy.core.Element
Returns:The element to be validated
